# Editorial Fashion Slider 

A Pen created on CodePen.

Original URL: [https://codepen.io/Margarita-the-solid/pen/NPRpXjB](https://codepen.io/Margarita-the-solid/pen/NPRpXjB).

A high-fashion editorial lookbook interface built as a pure HTML/CSS/JS single file. Features a structured 3-column grid layout with a custom lerp cursor, full-bleed image stage with Ken Burns transitions, filmstrip contact sheet, and a right panel with numbered inspo copy blocks. Typography system uses Cormorant Garamond (display serif), Unbounded (900wt), and Space Mono - mixing solid fill and stroke-outline headlines. Auto-advances with a gold progress line, keyboard/touch/click navigation, and responsive stacking on mobile.
